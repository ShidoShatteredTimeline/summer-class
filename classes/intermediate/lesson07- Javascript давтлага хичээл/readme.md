## Дасгал:

1. : Өгөгдсөн object төрөлтэй car хувьсагчийн бүх Properties болон утгуудыг буцаадаг функц бичнэ үү. Жишээ:

```sh
 car = {
  brand:'Toyota',
  name: 'Prius',
  price: 20000000
  }
  Гаралт:
  brand= Toyota
  name= Prius
  price= 2000000
```
